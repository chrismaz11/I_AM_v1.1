# I AM – Agent Suite (codex-cli)

This directory defines all autonomous and semi-autonomous LLM agents used to build, maintain, verify, and document the I AM platform.

Each agent uses the Goldilocks Framework:
1. Context 
2. Objective
3. Method
4. Constraints
5. Output Format
6. Checklist
